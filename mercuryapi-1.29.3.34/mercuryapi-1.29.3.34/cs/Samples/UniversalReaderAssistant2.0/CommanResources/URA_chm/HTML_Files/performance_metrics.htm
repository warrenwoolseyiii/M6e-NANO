Performance Metrics

Navigation Path: Settings/Status > Performance Metrics

This section displays performance metrics of the active or just performed Read operation:

Table 2: Performance Metrics

Setting Description
Unique Tag Count

Total number of unique tags found.
Total Tag Count

Total number of tags read, including re-reads of the same tag. Also displays the time during which those tags were read.
ReadRate/Sec

Using the Total Tag Count metric, displays the average tags per second.
