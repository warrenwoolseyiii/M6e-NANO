Apple Bonjour mDNS Discovery Service

Navigation Path: Installing URA > Apple Bonjour mDNS Discovery Service

URA2 uses Apple's Bonjour service to discover Bonjour enabled Network Readers. If you do not want Bonjour to be installed simply Cancel out of Bonjour installer once its appears during the URA install and it will not be installed. URA2 will continue to be installed even if Bonjour is cancelled.

If not installed the Network Readers list will be empty during Connect and you will need to manually enter the hostname or IP address.
