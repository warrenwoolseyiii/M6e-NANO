Source and Executable from MercuryAPI SDK

Navigation Path: Installing URA > Source and Executable from MercuryAPI SDK

In addition to being available as a standalone application URA, along with all of its source code, is part of the MercuryAPI SDK. URA can be accessed via the SDK as follows:

  1. Use a web browser to navigate to http://rfid.thingmagic.com/devkit
  2. Find the MercuryAPI SDK link under MercuryAPI | Software and click on it. This will begin download SDK Zip package.
  3. Once downloaded, double-click on the zip file to extract its contents.

Once the contents have been extracted the URA Visual Studio project can be found in:

[INSTALLDIR]/cs/Samples/UniversalReaderAssistant2.0

and the pre-built executable, along with the installer can be found in:

[INSTALLDIR]/cs/Samples/exe
