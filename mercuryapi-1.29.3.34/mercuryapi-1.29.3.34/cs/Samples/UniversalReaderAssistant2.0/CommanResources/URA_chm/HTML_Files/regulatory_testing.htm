Regulatory Testing

Navigation Path: Settings/Status > Regulatory Testing

This section is used to create test signals and customize the hop table to adhere to local regulations that limit the channel frequency range beyond limitations imposed by the region setting.

Note : These settings do not apply to M6 or Astra-EX readers. Their hop table can only be limited by editing a configuration file on the reader.

Three tabbed choices are available:

  • CW - will send an unmodulated carrier for a configurable amount of time (in msec). The channel used will be the first one in the hop table.
  • PRBS - Will send a signal modulated with random data for a configurable amount of time (in msec). The channel used will be the first one in the hop table.
  • Hop Table - Comma-separated (no spaces) list of all the channels in the hop table, in the order they will be used. Selecting a new region always refreshes the list to the default hop table for that region. ThingMagic modules support up to 62 entries in their hop table (no pre-defined regions currently have a hop table larger than 51 entries).
