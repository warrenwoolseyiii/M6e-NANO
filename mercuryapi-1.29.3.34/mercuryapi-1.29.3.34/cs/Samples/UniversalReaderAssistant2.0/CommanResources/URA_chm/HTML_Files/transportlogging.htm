Transport Logging

Navigation Path: Connect > Transport Logging

For both connection types if Transport Logging is checked a logfile will be created containing all of the low level communication between URA and the reader. This can be very important to diagnosing a support problem.

When gathering logs for support it is recommended the following process be followed:

  1. Prior to connect, check Transport Logging.
  2. Click Connect.
  3. Navigate to a filesystem location to save the log file and provide a name.
  4. Perform the operations to reproduce the problem being debugged.
  5. Once the problem has been reproduced, Disconnect.
  6. Send the log to support.
